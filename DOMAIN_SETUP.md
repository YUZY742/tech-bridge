# ドメイン設定ガイド

## 現状

現在、TECH-BRIDGEはVercelの無料ドメインで運用されています：
- **URL**: `https://tech-bridge-uybw.vercel.app`

## Vercel無料ドメイン vs 独自ドメイン

### Vercel無料ドメイン（現状）

**メリット：**
- ✅ **完全無料** - 追加コストなし
- ✅ **SSL証明書自動付与** - HTTPS対応済み
- ✅ **即座に利用可能** - 設定不要
- ✅ **CDN最適化済み** - 高速配信
- ✅ **自動更新** - 証明書の更新も自動

**デメリット：**
- ❌ **プロフェッショナル感に欠ける** - `.vercel.app`ドメインは開発/テスト感がある
- ❌ **ブランディングに不利** - 企業向けサービスとしては信頼性に疑問を持たれる可能性
- ❌ **SEOに不利** - カスタムドメインの方が検索エンジンで評価されやすい
- ❌ **メールアドレスが使えない** - `info@tech-bridge-uybw.vercel.app`は使えない
- ❌ **ブランド価値が低い** - 長期的なブランド構築には不向き

### 独自ドメイン

**メリット：**
- ✅ **プロフェッショナルな印象** - 企業向けサービスとして信頼性が高い
- ✅ **ブランディング** - 独自のブランドを確立できる
- ✅ **SEOに有利** - カスタムドメインの方が検索エンジンで評価されやすい
- ✅ **メールアドレス利用可能** - `info@tech-bridge.jp`などが使える
- ✅ **長期的な価値** - ブランド資産として蓄積される
- ✅ **企業からの信頼** - 大手企業との取引時に有利

**デメリット：**
- ❌ **コストがかかる** - 年間数千円〜数万円
- ❌ **設定が必要** - DNS設定など初期設定が必要
- ❌ **更新管理** - ドメインの更新を忘れないようにする必要がある

## 推奨事項

### 現段階（開発・テスト段階）

**Vercel無料ドメインで十分**

理由：
- プロトタイプ段階では無料で運用できる
- 機能開発に集中できる
- 本番運用前にドメインを検討できる

### 本番運用・企業向け公開時

**独自ドメインの取得を強く推奨**

推奨ドメイン例：
- `tech-bridge.jp` - 日本向けサービスに最適
- `tech-bridge.co.jp` - 企業向け（法人登録が必要）
- `techbridge.jp` - シンプル
- `techbridge.co.jp` - 企業向け

**理由：**
1. **企業からの信頼**
   - 大手製造業・重工業企業が支援を検討する際、`.vercel.app`ドメインでは信頼性に疑問を持たれる可能性
   - 独自ドメインは「本気度」を示す

2. **ブランディング**
   - TECH-BRIDGEというブランドを確立するには独自ドメインが必須
   - 長期的なブランド価値の構築

3. **マーケティング**
   - 名刺、資料、プレゼンでURLを共有する際、プロフェッショナルな印象
   - SEO対策としても有効

## ドメイン取得のタイミング

### 推奨タイミング

1. **ベータ版公開前**
   - 実際の企業にデモを見せる前
   - プレスリリース前

2. **最初の企業パートナー獲得前**
   - 正式な取引を開始する前
   - 契約書にURLを記載する前

3. **資金調達前**
   - 投資家にプレゼンする前
   - 補助金申請時

### 今すぐ取得すべき場合

- 既に企業との打ち合わせが始まっている
- プレスリリースを予定している
- 投資家へのプレゼンを予定している

## ドメイン取得の手順（Vercel連携）

### 1. ドメイン取得

**推奨レジストラ：**
- **お名前.com** - 日本で最も人気、日本語サポート
- **ムームードメイン** - シンプルで分かりやすい
- **Google Domains** - シンプルな管理画面（2023年9月以降はSquarespaceに移行）

**取得手順：**
1. レジストラでアカウント作成
2. 希望ドメインを検索
3. 購入（年間1,000円〜3,000円程度）
4. 支払い完了

### 2. Vercelでの設定

1. **Vercel Dashboardにアクセス**
   - https://vercel.com/dashboard

2. **プロジェクトを開く**
   - `tech-bridge-uybw`プロジェクトを選択

3. **Settings → Domains**
   - 「Add Domain」をクリック

4. **ドメインを入力**
   - 例: `tech-bridge.jp`
   - 「Add」をクリック

5. **DNS設定の指示に従う**
   - VercelがDNS設定方法を表示
   - レジストラのDNS設定画面で設定

**DNS設定例：**
```
Type: A
Name: @
Value: 76.76.21.21

Type: CNAME
Name: www
Value: cname.vercel-dns.com
```

6. **SSL証明書の自動発行を待つ**
   - 通常、数分〜数時間で完了
   - Vercelが自動でSSL証明書を発行

### 3. リダイレクト設定（オプション）

Vercelの無料ドメインから独自ドメインへリダイレクト：

`vercel.json`に追加：
```json
{
  "redirects": [
    {
      "source": "/(.*)",
      "destination": "https://tech-bridge.jp/$1",
      "permanent": true,
      "has": [
        {
          "type": "host",
          "value": "tech-bridge-uybw.vercel.app"
        }
      ]
    }
  ]
}
```

## コスト比較

### Vercel無料ドメイン
- **年間コスト**: 0円
- **初期設定**: 不要

### 独自ドメイン
- **ドメイン取得費**: 年間1,000円〜3,000円
  - `.jp`: 約3,000円/年
  - `.co.jp`: 約5,000円/年（法人登録が必要）
  - `.com`: 約1,500円/年
- **Vercelでの設定**: 無料
- **SSL証明書**: 無料（Vercelが自動発行）

**合計**: 年間1,000円〜5,000円程度
